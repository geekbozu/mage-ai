## Basic config

```yaml
connector_type: nats

server_urls: ['nats://demo.nats.io:4222']
subject_name: test
wait_time_seconds: 1
```

## Config with jetstream consumer

```
connector_type: nats

server_urls: ['nats://demo.nats.io:4222']
subject_name: test
wait_time_seconds: 1
# Jetstream will use a Pull based consumer, pulling in Batchsize, over wait_time_seconds
# Assumes you have a Durable created to bind to. 
jetstream_config:
    batch_size: 1   
    durable_name: test
    stream_name: test
```
